# Contributing to Q.Synapse

Thank you for considering contributing to the Q.Synapse project! We appreciate your interest.

## Development Setup Instructions

To get started with development, follow these steps:

### Prerequisites
- Ensure you have [Node.js](https://nodejs.org/) installed (version X.X.X or higher).
- Install [Git](https://git-scm.com/) if you haven't already.

### Step 1: Clone the Repository

```bash
git clone https://github.com/MVihma/Q.Synapse.git
cd Q.Synapse
```

### Step 2: Install Dependencies

```bash
npm install
```

### Step 3: Start Development Server

```bash
npm start
```

### Step 4: Make Your Changes
- Create a feature branch for your changes:

```bash
git checkout -b feature/your-feature-name
```

- Make your changes and commit them:

```bash
git add .
git commit -m "Add a meaningful commit message"
```

### Step 5: Push Changes and Create Pull Request

- Push your changes:

```bash
git push origin feature/your-feature-name
```

- Go to the repository on GitHub and create a pull request.

We look forward to your contributions! If you have any questions, feel free to reach out.
